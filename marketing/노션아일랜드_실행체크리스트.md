# 노션 아일랜드 스마트스토어 — 실행 체크리스트 (출력용)

> 전략 상세: [`노션아일랜드_스마트스토어_마케팅전략.md`](./노션아일랜드_스마트스토어_마케팅전략.md)
> 목표: **신규 매출·전환 확대** (유입 × 전환율 × 객단가)

## ✅ 이번 주 (비용 0원 — 전환·노출 기반)
- [ ] 상세페이지 **모바일 우선** 리뉴얼: 상단 3초 후킹 + 핵심혜택 3 + CTA
- [ ] 신뢰요소 박스: 배송일정·교환환불·인증·"누적판매 ○○개"
- [ ] **포토/동영상 리뷰 적립 이벤트** ON (포인트 차등)
- [ ] 배송완료 +3일 **리뷰요청 알림톡** 자동화
- [ ] 상품명/카테고리/태그 **SEO 최적화** (네이버 검색광고 키워드도구 활용)
- [ ] 네이버 톡톡 ON + 자주묻는질문 자동응답
- [ ] **알림받기 첫구매 쿠폰** 세팅 (신규 전환 트리거)

## ✅ 4~8주차 (유료 유입 점화)
- [ ] **쇼핑검색광고** 소액 테스트(일 1~3만원, 키워드 10~20개) → ROAS로 재배분
- [ ] 파워링크 대표키워드 + 브랜드 검색 방어
- [ ] **GFA 리타게팅**(방문·장바구니 이탈자) + 전환추적 픽셀 설치
- [ ] 블로그 주 1~2편(정보형) + **숏폼 주 3~5편**(릴스/유튜브 숏)
- [ ] 첫 **네이버 쇼핑라이브** 1회 (라이브 한정 쿠폰)

## ✅ 9~12주차 (확장·고정화)
- [ ] 효자 채널·소재 집중 증액, 지는 채널 컷
- [ ] 기획전 / 럭키투데이 / 도착보장 제안으로 노출면 확대
- [ ] 라이브 정기화(월 1~2회) + 체험단으로 우호 콘텐츠 증폭
- [ ] 알림받기·톡톡친구 **CRM 메시지 정례화** (재구매·재입고·시즌)
- [ ] **골라담기·여행지 컬렉션 세트·무료배송 임계금액(예 15,000원↑)** 으로 객단가 상승 (저가문구 필수)
- [ ] 다꾸 ASMR·비포애프터 숏폼 + 고객 다꾸사진 UGC 해시태그 이벤트

## ⚠️ 집행 전 반드시 (패널티·낭비 방지)
- [ ] 상품명 키워드 도배·허위 카테고리 금지 (검색누락·판매정지 위험)
- [ ] 전환 기반(상세·리뷰) 없이 광고비 선집행 금지
- [ ] 과장·허위 표현, 가짜 리뷰 금지 (신뢰도 제재)
- [ ] 초기엔 **쇼핑검색광고 + 숏폼 1~2채널 집중** (분산 과다 금지)

## 📊 매월 점검 지표
- [ ] 방문수 + 유입경로별 비중(검색/광고/SNS)
- [ ] 전환율(%) · 장바구니→결제 이탈률 · 채널별 **ROAS**
- [ ] 리뷰수·평점 · 찜·알림받기 증가수 · 톡톡 응답속도
- [ ] 객단가 · 재구매율 · 신규/재구매 매출 비중
