# 죽전에스치과 마케팅 실행 체크리스트 (출력용)

> 전략 상세: [`죽전에스치과_지역마케팅_전략.md`](./죽전에스치과_지역마케팅_전략.md)

## ✅ 이번 주 (비용 0원으로 당장 시작)
- [ ] 네이버 스마트플레이스: 정보·진료시간·**주차**·전화·예약링크 100% 채우기
- [ ] 네이버 스마트플레이스: 외관/대기실/진료실/장비/의료진 **고화질 사진** 다량 등록
- [ ] 대표 키워드 등록: 죽전치과 · 죽전역치과 · 모현치과 · 오포치과 · 용인임플란트 · 죽전소아치과
- [ ] 당근 비즈프로필 개설(무료) + 사진·진료안내 등록
- [ ] 카카오톡 채널 개설 (예약·리콜용)
- [ ] 접수지/태블릿에 **"어떻게 알고 오셨나요?"** 항목 추가 (유입경로 측정)

## ✅ 4~8주차 (타깃 점화)
- [ ] **모현 힐스테이트 몬테로이 + 오포 주요단지 엘리베이터 광고** 견적·집행 (※심의 확인)
- [ ] 당근 지역광고 ON — 모현읍·오포 동/단지 단위 타깃
- [ ] 제휴 미팅 5곳↑: 소아과·약국·단지 상가·학원·키즈카페
- [ ] 네이버 블로그 주 1~2편 (정보형: 충치예방/임플란트 Q&A/소아치과)
- [ ] 용인·광주·모현 입주민 맘카페 모니터링 (광고글 금지, 정직 응대만)

## ✅ 9~12주차 (확장·고정화)
- [ ] 어린이집/유치원 구강보건 캠페인 1회 (사전 협의 필수)
- [ ] 카톡채널 정기검진 리콜 발송 (이탈 방지)
- [ ] 지역별(모현/오포/죽전) 신환 데이터 분석 → 채널 예산 재배분

## ⚠️ 집행 전 반드시 (법적 체크)
- [ ] 현수막·전단·버스·옥외광고 → **의료광고 사전심의 접수** (대한치과의사협회)
- [ ] 모든 채널에서 **할인·이벤트·치료후기 광고·"최고/100%/안전" 단정표현** 사용 금지
- [ ] 심의 면제 조건 확인: 명칭·소재지·전화번호·진료과목만 = 심의 불필요

## 📊 매월 점검 지표
- [ ] 신환 수 + 유입경로별 분포
- [ ] 지역별 신환 비중(모현/오포/죽전)
- [ ] 플레이스 조회·전화·길찾기·예약 클릭 수
- [ ] 당근 노출·클릭·쿠폰사용
- [ ] 재방문율·리콜 응답률
