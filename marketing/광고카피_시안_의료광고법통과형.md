## 1. 아파트 엘리베이터 TV 광고 (모현·오포 신축단지)

엘리베이터 TV는 **5~15초 정지화면 또는 짧은 영상**이 일반적입니다. 글자는 적게, 크게. 한 화면에 핵심 1개.

### 시안 A — "접근성" 소구 (메인 추천)
```
[큰 글씨]   모현·오포에서 가까운 동네 치과
[중간]      죽전역 인근 · 주차 가능
[브랜드]    죽전에스치과
[하단고정]  용인시 수지구 정든로 5 (죽전역 인근) | ☎ 031-○○○-○○○○ | 진료과목: 치과
```
*비주얼: 밝고 깨끗한 진료실 또는 외관 사진 1컷, 따뜻한 톤.*

### 시안 B — "가족" 소구 (자녀 둔 입주 가구 타깃)
```
[큰 글씨]   온 가족이 함께 다니는 치과
[중간]      아이부터 부모님까지, 정성껏 진료합니다
[브랜드]    죽전에스치과
[하단고정]  용인시 수지구 정든로 5 (죽전역 인근) | ☎ 031-○○○-○○○○ | 진료과목: 치과
```
*비주얼: 가족 친화적 이미지(연출 사진), 키즈 공간 있으면 노출.*

### 시안 C — "신규 입주민 환영" 소구 (입주 6~18개월 단지)
```
[큰 글씨]   새 동네, 첫 치과 검진은?
[중간]      이사 오신 모현·오포 이웃을 환영합니다
[브랜드]    죽전에스치과 — 구강검진 상담 안내
[하단고정]  용인시 수지구 정든로 5 (죽전역 인근) | ☎ 031-○○○-○○○○ | 진료과목: 치과
```

### 시안 D — [심의면제형] 정보 고지 only (가장 안전)
```
[로고/상호]  죽전에스치과
[정보]       진료과목: 치과
            용인시 수지구 정든로 5, 203호 (죽전역 인근)
            ☎ 031-○○○-○○○○
            진료시간 평일 09:30–18:30 (점심 12:30–14:00)
            주차 가능
```
> 슬로건·권유 문구가 전혀 없으므로 사전심의 없이 게재 가능. 인지도 확보용으로 부담 없이 사용.

아이부터 어르신까지 가족 모두 편안하게 진료받으실 수 있도록

> ⚠️ 당근 본문에도 **가격·할인·"무통/최고/100%"** 표현 금지. 진료과목 "고지"와 접근성·진료철학 위주로.

### 단골 소식 게시글 (월 2~4회, 광고 티 X)

- "아이 첫 치과 방문, 언제가 좋을까요?"
- "스케일링, 얼마나 자주 받아야 할까요?"
- "사랑니, 꼭 빼야 하나요? 빼야 하는 경우 정리"
- "임플란트 전에 알아두면 좋은 기본 상식"

각 글 말미 공통 마무리:

## 3. 당근 지역광고 소재 (유료 — 모현읍·오포 동/단지 타깃)

당근 광고는 **이미지 1장 + 짧은 카피 + 가게이름/위치**가 자동 노출됩니다. 카피만 준비하면 됩니다.

### 광고 카피 A/B 테스트 세트
| 버전 | 헤드라인 (≤ 20자 권장) | 서브 |
|---|---|---|
| A (접근성) | 모현·오포에서 가까운 치과 | 죽전역 인근 · 주차 가능 |
| B (가족) | 온 가족 다니는 동네 치과 | 아이부터 부모님까지 |
| C (입주민) | 새 동네 첫 치과 검진 | 모현·오포 이웃 환영합니다 |
| D (편안함) | 편안하게 진료받는 치과 | 정성껏 살펴드립니다 |

> A안과 C안을 먼저 동시 집행 → 클릭·전화 데이터로 우세안 선택 후 예산 집중.

## 4. 이미지/디자인 가이드 (촬영·제작 지침)

엘리베이터·당근 공용 에셋. **사진 품질이 클릭률을 좌우**합니다.

### 꼭 찍어야 할 컷 (촬영 체크리스트)
- [ ] **외관/간판** (낮·밝게) — 위치 신뢰감
- [ ] **대기실** (밝고 깨끗, 정돈) — 첫인상
- [ ] **진료실/체어** (청결, 장비)
- [ ] **소독·멸균 공간** (위생 신뢰 — 사실 기반이라 안전한 소구)
- [ ] **의료진/직원** (밝은 표정, 환자 동의 없는 환자 노출 금지)
- [ ] **주차장/주차 안내** — 차로 오는 모현·오포 환자에 중요
- [ ] **키즈존**(있으면) — 가족 타깃

### 톤 & 매너
- **컬러:** 화이트 + 민트/블루 톤(청결·신뢰). 과한 원색·자극적 디자인 금지.
- **폰트:** 굵고 큰 산세리프(엘리베이터는 멀리서도 읽혀야 함). 한 화면 글자 수 최소화.
- **금지 비주얼:** 시술 전후 비교 사진(❌ 심의 위험), 과장된 그래프, "▼50%" 류 할인 띠.
- **로고/상호**는 항상 동일 위치에 고정(인지도 누적).

| 매체 | 추천 시안 | 심의 | 비고 |
