### 이 프로필이 전략에 주는 3가지 결정적 함의
1. **객단가가 매우 낮다(1,500~8,000원).** 단품으로 팔면 **배송비가 상품가보다 비싸** 수익이 안 난다.
   → **"골라담기/묶음/세트 + 무료배송 임계금액"이 수익성의 핵심.** (단품 구매를 막고 장바구니를 키운다)
2. **다꾸·여행 = 숏폼 황금 소재.** 마테 뜯어쓰기·스티커 붙이기·다꾸 과정은 **틱톡/릴스에서 가장 잘 터지는 포맷.**
3. **SKU 133종 = 빠른 신상 회전.** 낱개 나열보다 **여행지/테마 "컬렉션·시리즈"로 묶어** 브랜딩·검색·세트판매에 활용.

## 0. 한 장 요약 — "SNS는 허브, 스토어는 계산대"

```
        [콘텐츠 허브]                      [판매 채널 = 계산대]
   ┌──────────────────┐            ┌──────────────────────────────┐
   │  인스타 릴스/피드  │ ── 유입 ──▶│ 아이디어스  (발견·핸드메이드 팬)  │
   │  틱톡 숏폼(신규)   │            │ 스마트스토어 (검색·메인 전환)     │
   └──────────────────┘            │ 교보문고   (신뢰·선물·오프라인)   │
            ▲  콘텐츠 1개 → 다채널 재활용  └──────────────────────────────┘
            └─────────── 링크인바이오(원페이지)로 동선 통일 ───────────┘
```

핵심 원칙 3가지
1. **콘텐츠는 한 번 만들어 여러 채널에 재활용** (틱톡 영상 → 인스타 릴스 → 유튜브 숏 → 스토어 상세)
2. **채널마다 역할이 다르다** — 같은 가격·같은 구성으로 다 깔지 말 것
3. **모든 SNS 동선은 "링크인바이오 원페이지"로 통일** → 거기서 채널 선택 유도

## 1. 판매 채널 지도 — 역할 분담

3채널을 "다 똑같이"가 아니라 **고객·목적별로 다르게** 운영해야 카니발라이제이션(자기잠식)을 막는다.

| 채널 | 고객 성격 | 역할 | 운영 포인트 |
|---|---|---|---|
| **아이디어스** | 핸드메이드·작가 팬, 취향소비, 충성도 높음 | **팬덤·단골** 형성, 작가 스토리텔링 | 작가 소식·후기 적극 활용, 한정/주문제작·클래스 가능, 소통이 곧 노출 |
| **스마트스토어** | 네이버 검색 유입, 가격·후기 비교 | **메인 전환·검색 유입·광고 거점** | SEO·리뷰·쇼핑검색광고 집중(별도 전략문서), 객단가·재구매 설계 |
| **교보문고 온라인** | 도서·문구 구매층, 선물 수요, 브랜드 신뢰 중시 | **신뢰·선물·확장 매출**, 오프라인 연계 가능성 | MD 협업·기획전, 도서/다이어리 시즌(연말·신학기) 공략, 패키지·바코드·납품규격 준비 |

### 1-1. 채널별 가격·구성 전략 (자기잠식 방지)
- **동일 상품을 3곳에 똑같이 깔지 말 것.** 채널별로 차별 포인트를 둔다.
  - 아이디어스: **한정판/주문제작/작가 시그니처 세트** (희소성·스토리)
  - 스마트스토어: **골라담기 + 여행지 컬렉션 세트** (검색·가성비·재구매)
  - 교보문고: **선물용 패키지/스페셜 에디션** (선물 수요·브랜드감)
- 채널별 수수료가 다르므로 **마진표를 채널별로 따로** 만들고 가격·프로모션 설계
- 한 채널 품절/배송지연이 다른 채널 평판으로 번지지 않게 **재고·CS 기준 통일**

### 1-2. ⭐ 객단가 설계 — 저가 문구의 생존 공식 (가장 중요)
상품이 1,500~8,000원이라 **단품 배송은 적자**다. 무조건 장바구니를 키워야 한다.
- **골라담기(N개 묶음 할인):** "스티커·마테 자유롭게 5개 담으면 10%" 식으로 단품 구매 차단
- **무료배송 임계금액:** 예) **15,000원↑ 무료배송** → 추가 담기 유도(임계선은 마진 계산해 설정)
- **여행지 컬렉션 세트:** `엽서+스티커+마테` 도시 테마 묶음 = 객단가 ↑ + 선물 수요 흡수
- **다꾸 키트/스타터팩:** 입문자용 모음 구성으로 첫 구매 장벽 낮추고 객단가 상승
- 동봉 전략: 주문에 **미니 스티커/엽서 1장 동봉 + "다음 구매 쿠폰"** → 재구매·후기 유도

### 1-3. 교보문고 입점 — 별도 준비 체크 (입점 예정이므로 지금 준비)
- [ ] **상품 규격 정비:** 자체 바코드, 낱개 포장, 패키지 디자인, 납품 단가/수수료 구조 확인
- [ ] **MD 제안 자료:** 브랜드 소개·베스트 상품·SNS 반응·기존 채널 판매실적 1장 요약 (입점/기획전 제안용)
- [ ] **여행 문구 = 핫트랙스 궁합 우수:** 교보 핫트랙스의 여행/다이어리/다꾸 코너 MD에 시리즈로 제안
- [ ] **시즌 캘린더 정렬:** 다이어리/문구는 **연말~신학기(11~3월)** + **여름 휴가철(여행 테마!)** 이 성수기 → 입점·물량·프로모션 타이밍 맞추기
- [ ] **온·오프 연계 타진:** 온라인 반응 좋으면 핫트랙스 오프라인 입점·팝업 제안 루트 확인
- [ ] 교보 입점 사실 자체를 **SNS·기존 채널에 "교보문고 입점!" 콘텐츠로 홍보** → 브랜드 신뢰 증폭 (입점이 곧 마케팅 소재)

## 2. 인스타그램 운영방향 — 700 팔로워 → 다음 단계

3년 운영에 700명이면 **피드 중심으로 정체**됐을 가능성이 크다. 지금 인스타 성장 엔진은 **릴스(발견 탭)** 다.

### 2-1. 진단 → 전환 포인트
- 피드(정적 사진) 위주 → **릴스 중심으로 무게추 이동** (도달·신규 유입은 릴스에서 나온다)
- 팔로워 700명에 매몰되지 말 것 — **저장(Save)·공유(Share)·도달**이 더 중요한 성장 지표
- 프로필을 "구매 동선"으로: 한 줄 소개에 **무엇을 파는지 + 링크인바이오** 명확히

### 2-2. 콘텐츠 기둥 (4가지를 돌려가며)
1. **다꾸 과정** — 엽서·스티커·마테로 다이어리 한 페이지 완성하는 영상 (저장·따라하기 유발 = 도달 폭발)
2. **여행 감성 무드** — 도시 시리즈 신상을 여행 사진/플레이리스트와 엮은 무드 콘텐츠 (브랜드 결)
3. **제작·비하인드·작가 스토리** — 만드는 과정, 입점 소식(교보문고!), 여행지 컬렉션 비하인드 → 팬덤
4. **고객·후기 리그램** — 고객의 다꾸 사진 리그램 = 신뢰 + 콘텐츠 + UGC 확보 (해시태그 이벤트로 수집)

### 2-3. 운영 규칙
- **릴스 주 3~4편 + 피드/캐러셀 주 1~2편 + 스토리 거의 매일**(투표·질문스티커로 소통)
- 릴스는 **첫 1초 후킹 + 자막 + 트렌드 사운드 + 7~15초** (짧고 빠르게)
- 하이라이트로 정리: `제품` `후기` `구매방법(채널별)` `브랜드`
- 해시태그: 대형(#문구스타그램 #다꾸) + 중형(#다이어리꾸미기 #마스킹테이프 #여행스티커) + 소형/브랜드(#노션아일랜드) 섞어 10~15개
- **공동구매·작가 협업·릴스 콜라보**로 새 팔로워층 유입 (비슷한 결의 문구·취향 계정과)

## 3. 틱톡 신규 채널 — 0 → 1 런칭 플랜

틱톡은 팔로워 0이어도 **발견(For You) 도달이 커서 신규 브랜드에 유리**하다. 핸드메이드 제작과정은 틱톡에서 특히 잘 먹힌다.

### 3-1. 셋업 (개설 첫 주)
- [ ] 비즈니스 계정 전환, 프로필·링크(링크인바이오) 세팅, 인스타와 핸들 통일
- [ ] 인스타 기존 릴스 중 반응 좋았던 것 **5~10개를 틱톡 포맷으로 리업로드** (무에서 시작하지 않기)
- [ ] **TikTok Shop / 공식 스토어 링크** 연결 가능 여부 확인 → 영상에서 바로 구매 동선

### 3-2. 콘텐츠 포맷 (틱톡에서 잘 되는 것)
- **다꾸 ASMR / 타임랩스** — 마테 뜯는 소리, 스티커 떼붙이는 소리, 빈 페이지→완성 (저가 문구의 최강 무기)
- **언박싱·패키징** 주문 포장 영상 ("같이 포장해요", 동봉 굿즈 보여주기)
- **트렌드 사운드/챌린지 올라타기** + 신상 스티커·마테 자연스럽게 노출
- **비포→애프터**(빈 다이어리 → 노션아일랜드 굿즈로 꾸민 다이어리) — 변화가 클수록 잘 터짐

### 3-3. 운영 규칙
- **주 4~7편**(틱톡은 빈도가 곧 도달) — 인스타 릴스와 **같은 소재 재활용**해서 제작부담 최소화
- 영상마다 자막·후킹·CTA(프로필 링크 안내)
- 잘 터진 영상은 **틱톡 광고(소액) + 인스타 릴스로 교차 증폭**
- 댓글에 적극 응대 → 노출 알고리즘에 유리

## 4. 콘텐츠 운영 시스템 — "1번 찍어 여러 번 쓴다"

채널이 5개(인스타·틱톡·3개 스토어)라 따로 만들면 지친다. **배치 제작 + 재활용**이 생존 전략.

### 4-1. 1콘텐츠 → 다채널 재활용 흐름
```
세로 영상 1개 촬영
   ├─▶ 틱톡 (원본)
   ├─▶ 인스타 릴스
   ├─▶ 유튜브 숏 (선택)
   ├─▶ 사진 캡처 → 인스타 피드/캐러셀
   └─▶ 스토어 상세페이지 + 아이디어스 소식 + 교보 상세
```

### 4-2. 주간 운영 루틴(예시)
- **월:** 한 주치 촬영 배치(제작과정·연출·언박싱 몰아 찍기)
- **화~금:** 편집·예약발행(릴스/틱톡), 스토리 데일리
- **수·금:** 후기 리그램, 댓글·DM·톡톡 응대
- **금:** 주간 지표 확인 → 잘된 영상 광고 증폭

- 교보문고 입점 완료 → "교보 입점!" 콘텐츠로 전 채널 홍보
- 아이디어스: 한정/주문제작 + 후기 이벤트로 단골화
- 작가/계정 협업·공동구매 1건 실행 (신규 팔로워 유입)

**61~90일 — 확장·정렬**
- 채널별 KPI 리뷰 → 효율 채널·포맷에 자원 집중
- 시즌(연말·신학기) 선물 패키지 기획 — 교보·스마트스토어 동시 푸시
- SNS→스토어 전환 동선 최적화, 재구매·CRM(알림받기·아이디어스 소식) 정례화
- 다음 분기: 오프라인(핫트랙스/팝업) 입점 타진 여부 판단

## 7. 하지 말 것
